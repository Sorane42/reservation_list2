#  Système de Réservation d'Objets

Une application Full-Stack moderne pour gérer un catalogue d'objets, visualiser leur disponibilité et effectuer des réservations avec une date de fin.

> **Note :** Ce projet a été réalisé dans un but pédagogique pour s'entraîner au développement Full-Stack (React & PHP).

##  Fonctionnalités
- **Affichage dynamique** : Liste des objets avec badges de statut (Disponible/Réservé).
- **Ajout d'objets** : Formulaire via une fenêtre modale pour enrichir le catalogue.
- **Réservation interactive** : Sélection d'une date de fin via un calendrier pour bloquer un objet.

##  Stack Technique
- **Frontend** : React.js, Bootstrap 5, React-Bootstrap
- **Backend** : PHP 8
- **Base de données** : MySQL
